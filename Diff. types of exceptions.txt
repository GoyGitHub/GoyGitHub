Different types of exceptions in python.

*SyntaxError - mispelled keyword, missin colon or unbalanced parenthesis.
*TypeError - an object or the wrong type  e.g: adding string to integer.
*NameError - variable or function name is not found.
*IndexError - index out of range.
*KeyError - when key is not found in a dictionary.
*ValueError - an invalid or argument  e.g: converting a string.
*AttributeError - an attribute or method is not found on an object.
*IDError - an I/O operation fails due to an input/output error.
*ZeroDivisionError - diving a number by zero.
*ImportError - import statement fails due to find or load a module.
